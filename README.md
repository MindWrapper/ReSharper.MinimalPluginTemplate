Visual studio solution template for ReSharper plugin with tests. 
To install:

`dotnet new --install ReSharper.MinimalPluginTemplate`

`dotnet new ReSharper.MinimalPlugin --name ReSharper.MyPlugin --output <Path_To_Folder>`
